be careful when we set " DB2ENVLIST" 
As instance owner, run these sequence of commands on the db2 server.

1) export LDAP_DEBUG=65535
2) export LDAP_DEBUG_FILE=/tmp/ldap_client_svt5ods.out
3) db2set DB2ENVLIST="ICC_TRNG ICC_IGNORE_FIPS DB2_JVM_STARTARGS IBM_HEAPDUMPDIR IBM_JAVACOREDIR IBM_COREDIR LDAP_DEBUG LDAP_DEBUG_FILE"
4) ls -l /tmp/ldap_client_svt5ods.out
5) db2stop
6) db2start

TEST LDAP user connection. In this example the user is connecting to the ODS database.  A test of HADR would be using SALECONN.

7b) disconnect SCIWSTG
